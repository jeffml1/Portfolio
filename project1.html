Matthew Jeffery - Project 1

For Project 1 we are working on the Internet of Things. This project focuses on the inter-networking of devices such as smart devices, sensors, software, and other items embedded with electronics.

Connecting these devices to a network creates the Internet of Things and enables the collection and exchange of data between devices and server.

The IoT uses slightly different technology than WiFI to achieve this goal. Our IoT solution is implemented using LoRaWAN. The idea is to research and develop an open LoRaWAN consisting of one or more gateway devices, multiple sensor devices, and a central server. All of this in the Dunedin area, to provide access to user data for use in custom mobile/web/desktop applications.

We want to record the data retrieved from our multiple sensors into a database to build statistical graphs to cross reference the data with the location of the sensors
